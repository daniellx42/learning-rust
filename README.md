## Cargo
```bash
# execute your project with
$ cargo run
#compile your project with
$ cargo build
#test your project with
$ cargo test
#manage your project documentation with
$ cargo doc
#publish your library in crates.io with
$ cargo publish
#build with optimizations
$ cargo build --release
#help
$ cargo -h
#list of commands
$ cargo --list
```
